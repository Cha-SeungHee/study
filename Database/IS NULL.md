#### 이름이 없는 동물의 아이디 - IS NULL
``` SQL
SELECT ANIMAL_ID 
FROM ANIMAL_INS 
WHERE NAME IS NULL 
ORDER BY ANIMAL_ID
```

#### 이름이 있는 동물의 아이디 - IS NULL
``` SQL
SELECT ANIMAL_ID
FROM ANIMAL_INS
WHERE NAME IS NOT NULL
ORDER BY ANIMAL_ID
```

#### NULL 처리하기 - IFNULL
``` SQL
SELECT ANIMAL_TYPE, IFNULL(NAME, "No name") AS NAME, SEX_UPON_INTAKE
FROM ANIMAL_INS
ORDER BY ANIMAL_ID
```
- IFNULL 뒤에 AS NAME을 붙이지 않으면 IFNULL로 표출된다
